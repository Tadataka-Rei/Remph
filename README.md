## ✨ About

**Remph** is a personal calendar and schedule management application designed to help organize everyday life.

Whether it's planning the week ahead, remembering important events, or receiving reminders when the time comes, Remph quietly keeps track of your schedule so you can focus on what matters.

Unlike large productivity suites, Remph is intentionally simple—built for individuals who just need a reliable companion for managing their time.

---

Like the sea patiently smoothing stone, time quietly shapes our lives one moment at a time.

**Remph** is imagined as the *Angel of Time*—not a mythical being, but a silent guardian that keeps your plans, memories, and commitments close at hand.

---

## 🚀 Features

- 📅 Weekly planner
- 🗓️ Calendar view
- 📝 Create and manage events
- ⏰ Notifications and reminders
- 📌 Personal task planning
- 📖 Event notes
- 💾 Persistent PostgreSQL storage
- 🐳 Dockerized deployment

## ⚡ Getting Started

Clone the repository

```bash
git clone https://github.com/Tadataka-Rei/Remph
cd remph
```

Run everything with Docker

```bash
docker compose up
```

Default URLs

| Service  | URL                   |
| -------- | --------------------- |
| Frontend | http://localhost:3000 |
| Backend  | http://localhost:5000 |
